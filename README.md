Este repositório contém os artefatos produzidos nos dois projetos da aula, desenvolvidos utilizando técnicas de engenharia de prompt e iteração com IA.

- `projeto_1/`: Contém os arquivos e a lógica do primeiro projeto.
- `projeto_2/`: Contém os arquivos e a lógica do segundo projeto.

## Metodologia
Todo o código foi gerado, ajustado e depurado através de prompts em linguagem natural, mantendo a integridade do processo de desenvolvimento assistido.
